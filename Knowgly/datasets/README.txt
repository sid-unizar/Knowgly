You can use the provided scripts to download and convert the DBpedia 2015-10 and QUIRA imdb datasets to HDT.

Please read the comments included on each script beforehand.

Requirements:
    - A valid hdt-java-package installation directory accessible from here.
      We used hdt-java-package-3.0.9 (https://github.com/rdfhdt/hdt-java/releases/tag/v3.0.9)
